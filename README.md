## 주요 기능

### 기본적인 커뮤니티 기능
- 커뮤니티에서 사용되는 기본적인 기능들의 구현
- 회원가입, 로그인, 게시글 crud 등이 포함

### 게시글 지원자 현황
- 게시글에 누가, 얼마나 지원했는지 표시
- 이를 통해 오픈채팅에 들어가지 않고 인원을 파악할 수 있음

### 게시글 필터링

- 게시글을 검색할 때 기술스택을 통해 필터링 가능
- 이외에도 제목이나 작성자 등 기본적인 필터링 구현
